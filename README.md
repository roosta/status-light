Status Light
===

Control program for a 4x4 NeoPixel LED matrix framed and hanging next to my PC

## Permissions

Added add udev rules for boards here: `/etc/udev/rules.d/99-platformio-udev.rules`

## LSP

uv run pio run -t compiledb
